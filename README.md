# Pharmaceutical Organic Chemistry II - MCQ Web App

A comprehensive interactive multiple-choice question bank based on the provided PDF (240+ MCQs).  
Topics: SNAr, Pericyclic reactions, Heterocyclic compounds, Carbohydrates, Lipids, Proteins, Nucleic acids, Drug synthesis reagents, and more.

## Features

- **240+ high-yield questions** with accurate answers.
- **Filter by topic** (SNAr, Unsaturated carbonyls, Pericyclic, Heterocycles, etc.)
- **Instant scoring** after submission.
- **Reveal correct answers** for learning.
- **Responsive design** – works on mobile and desktop.
- **Clean file structure** ready for GitHub and deployment via median.co or any static host.

## How to Use

1. Clone or download the folder.
2. Open `index.html` in any modern browser.
3. Select answers by clicking on options.
4. Press **Submit & Score** to evaluate.
5. Use **Filter** to focus on specific topics.
6. Click **Reveal Correct** to see correct answers (even before submitting).

## Deployment

- **GitHub Pages**: Upload the folder to a GitHub repo, enable Pages.
- **median.co**: Package as a web app (PWA capabilities) or simply point to the `index.html`.

## Data Integrity

All questions and answers are extracted directly from the provided MCQ PDF (Pharmaceutical Organic Chemistry II). The dataset covers SNAr mechanisms, unsaturated carbonyls, pericyclic reactions, neighboring group effect, transition metal catalysis, stereoselectivity, heterocycles, carbohydrates, lipids, proteins, nucleic acids, drug structures, and named reactions.
